# synopsis 

 Free CI service for Raku opensource projects

# Add new project

```
sparrowdo \
--localhost \
--tags repo=gh-grondilu-pbkdf2-raku,\
git=https://github.com/grondilu/pbkdf2-raku.git,\
base=$PWD,\
image=debian_arm \
--no_sudo
```
